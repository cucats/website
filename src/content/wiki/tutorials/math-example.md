## Inline Math

You can write inline math like $E = mc^2$ or the quadratic formula $x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$ directly in your text.

## Display Math

$$
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
$$

## Code Example

```python
def factorial(n):
    if n <= 1:
        return 1
    return n * factorial(n - 1)
```
